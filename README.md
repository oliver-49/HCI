# School Admin Dashboard

**School Admin Dashboard** هو مشروع تم تصميمه باستخدام **Power BI** لإدارة وتحليل بيانات المدرسة بشكل متكامل.  
يساعد على تتبع أداء الطلاب، حضورهم، ومعلومات الإدارة المدرسية بطريقة سهلة ومرئية.

---

## Project Overview

Dashboard يحتوي على **4 Cards رئيسية**:
1. عدد الطلاب  
2. عدد المدرسين  
3. عدد الفصول  
4. نسبة الغياب

ويعرض أيضاً مجموعة من التحليلات والإحصائيات المتعلقة بالمدرسة والإدارة، مثل:

- **Students by Class**: توزيع الطلاب حسب الفصول  
- **Monthly Attendance Trend**: متابعة حضور الطلاب شهرياً  
- **Attendance Summary**: ملخص الحضور لكل طالب  
- **Average Grades by Subject**: متوسط درجات الطلاب لكل مادة  
- **Student Performance**: ترتيب الطلاب من حيث الأداء الأكاديمي  
- **Dynamic Filters**: اختيار الفصل، المادة، أو المدرس لعرض البيانات الخاصة بهم

---

## Files in Repository

- `School_Admin_Dashboard_1.pbix` : ملف Power BI الرئيسي للـ Dashboard  
- `school_data.xlsx` : بيانات الطلاب، حضورهم، والدرجات  
- `images/` : صور توضيحية لدعم الـ Dashboard

---

## Features

- عرض **Top Performing Students** حسب درجاتهم  
- تتبع **الحضور والغياب** لكل طالب  
- تحليل **الدرجات حسب المادة والفصل**  
- عرض **إحصائيات المدرسة والإدارة**  
- **Interactive Filters** لاختيار Class, Subject, أو Teacher  

---

## How to Use

1. افتح **School_Admin_Dashboard_1.pbix** في Power BI Desktop  
2. تأكد من استيراد البيانات من `school_data.xlsx`  
3. استخدم الـ **Slicers** لاختيار الفصول أو المواد أو المدرسين  
4. شاهد التحليلات والـ Cards تتحدث تلقائياً حسب اختيارك
